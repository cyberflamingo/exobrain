# Mutability vs Reassignment


## Mutability

```
                             |
                             |  +---+             +------------+
  a = 'Hello'                |  | a |  -------->  | 'Hello'    |
                             |  +---+             +------------+
                             |
-----------------------------+------------------------------------------
                             |
                             |  +---+
                             |  | a |  ---+
                             |  +---+      \
                             |              \     +---------------+
  b = a                      |               +--> | 'Hello'       |
                             |              /     +---------------+
                             |  +---+      /
                             |  | b |  ---+
                             |  +---+
                             |
-----------------------------+------------------------------------------
                             |
                             |  +---+
                             |  | a |  ----+
                             |  +---+       \     +---------------+
  a << ' there'              |               +--> | 'Hello there' |
                             |  +---+       /     +---------------+
                             |  | b |  ----+
                             |  +---+
                             |
-----------------------------+------------------------------------------
```


## Reassignment

```
                             |
                             |  +---+             +------------------+
  a = 'Hello'                |  | a |  -------->  | 'Hello there'    |
                             |  +---+             +------------------+
                             |
-----------------------------+------------------------------------------
                             |
                             |  +---+
                             |  | a |  ---+
                             |  +---+      \
                             |              \     +------------------+
  b = a                      |               +--> | 'Hello there'    |
                             |              /     +------------------+
                             |  +---+      /
                             |  | b |  ---+
                             |  +---+
                             |
-----------------------------+------------------------------------------
                             |
                             |  +---+             +------------------+
                             |  | a |  -------->  | 'General Kenobi' |
                             |  +---+             +------------------+
  a = 'General Kenobi'       |
                             |  +---+             +------------------+
                             |  | b |  -------->  | 'Hello there'    |
                             |  +---+             +------------------+
                             |
-----------------------------+------------------------------------------
```

Source: [Variable References and Mutability of Ruby Objects](https://launchschool.com/blog/references-and-mutability-in-ruby)
